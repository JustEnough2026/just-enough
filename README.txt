食材有限 Just Enough｜正式版點餐網站

使用方式：
1. 解壓縮後，直接開啟 index.html 可預覽。
2. 免費上線：把整個資料夾拖曳到 Cloudflare Pages、Netlify Drop 或 GitHub Pages。
3. 網站會在手機瀏覽器保存購物車，並可加入主畫面。
4. 「複製訂單並開啟 LINE」會先整理訂單文字，再打開 LINE 分享頁。

注意：
- 這是免費的純前端網站，不含雲端後台或資料庫。
- 今日公告、價格與品項可在 app.js / index.html 修改。
- LINE 按鈕由顧客自行在 LINE 內選擇要傳送的聊天室。

本版本已移除所有奇美醫院限定內容。
